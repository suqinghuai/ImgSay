## 📖 项目介绍

**图言 ImgSay** 是一个强大的AI驱动的图片文字识别工具，专门设计用于高效、准确地从图片中提取文字内容。相比传统OCR软件，图言利用先进的多模态AI模型，提供更高的识别准确率和更智能的文字理解能力。

### ✨ 核心优势

- 🎯 **超高准确率**：采用Qwen3-VL-8B-Instruct先进多模态模型，识别准确率远超传统OCR
- 🚀 **批量处理**：支持同时处理多张图片，大幅提升工作效率
- 🔄 **智能排序**：自动按自然顺序处理图片，确保结果连贯性
- 💾 **结果导出**：识别结果自动保存到文本文件，方便后续使用
- 🛠️ **灵活配置**：支持自定义AI模型、批量大小等参数
- 🆓 **免费使用**：ModelScope平台提供每日2000次免费调用额度

## 🎯 适用场景

- 📚 文档数字化：将纸质文档、书籍页面快速转换为可编辑文本
- 📊 数据提取：从图表、报表图片中提取关键数据信息
- 📝 笔记整理：识别手写笔记、会议记录等图片内容
- 🌐 多语言识别：支持中文、英文等多种语言文字识别
- 🏢 办公自动化：批量处理工作中的图片文字识别需求

## 🚀 快速开始

### 环境要求

- Python 3.12 或更高版本
- Windows/macOS/Linux 操作系统
- 稳定的网络连接（用于API调用）

### 安装步骤

1. **下载安装包**
   - 下载最新版本的 ImgSay.exe 可执行文件
   - 确保程序目录结构完整

2. **获取API密钥**
   - 访问 ModelScope官网 注册账号
   - 完成实名认证，绑定阿里云账号（认证后获得免费调用额度）
   - 在个人中心获取API Key

3. **配置API密钥**
   ```ini
   # 编辑 config.ini 文件
   API_KEY = your_api_key_here
   ```

4. **准备图片**
   - 将需要识别的图片放入 图片/ 文件夹
   - 支持格式：jpg、jpeg、png、webp、bmp（建议使用jpg格式，其它格式尚未经过严格测试）

5. **运行程序**
   - 双击运行 ImgSay.exe
   - 程序自动处理并生成识别结果

## ⚙️ 配置说明

### 配置文件 (config.ini)

```ini
[ModelScope]
# ModelScope API密钥（必填）
API_KEY = your_api_key_here

# API接口地址（通常无需修改）
API_URL = https://api-inference.modelscope.cn/v1/chat/completions

# 使用的AI模型
MODEL_ID = Qwen/Qwen3-VL-8B-Instruct

# 图片文件夹路径
IMG_FOLDER = 图片/

# 识别结果保存文件
RESULT_FILE = 识别结果.txt

# 每次处理的图片数量（建议1-8）
BATCH_SIZE = 4
```

### 推荐AI模型

| 模型名称 | 特点 | 适用场景 |
|---------|------|---------|
| Qwen/Qwen3-VL-8B-Instruct | 默认模型，多语言支持，准确率高 | 通用文字识别 |

## 🔧 技术实现

### 核心功能模块

1. **配置加载模块** (load_config())
   - 读取和验证配置文件
   - 检查API密钥和图片文件夹

2. **图片处理模块** (get_image_base64())
   - 将图片转换为Base64格式
   - 支持多种图片格式

3. **批量处理模块** (process_batch())
   - 调用AI模型API进行文字识别
   - 实现重试机制和错误处理
   - 保存识别结果到文件

## ❓ 常见问题

### Q1: 如何获取免费的API调用额度？
A: 在ModelScope完成实名认证后，每天可获得2000次免费调用，单个模型理论上不超过500次。

### Q2: 支持哪些图片格式？
A: 支持jpg、jpeg、png、webp、bmp等常见图片格式。

### Q3: 识别结果不准确怎么办？
A: 可能原因包括：
- API密钥无效或已过期
- 图片质量太差，文字不清晰
- 图片中文字太小或太密集
- 网络连接不稳定

### Q4: 可以离线使用吗？
A: 目前需要网络连接调用云端AI模型，暂不支持完全离线使用。

### Q5: 如何切换其他AI模型？
A: 只需在config.ini中修改MODEL_ID配置项即可切换到其他支持多模态的模型。
